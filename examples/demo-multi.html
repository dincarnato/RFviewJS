RFview.js · Multiple Alternative Structures

Animated transitions · per-structure labels · same sequence

Select structure

Animated transitions Switching between structures with the same sequence triggers a smooth morphing animation. Duration is configurable per viewer instance.
Structure switcher The pill bar at the top of the viewer also lets you switch — and is updated automatically when you click the sidebar buttons.

API usage

// Pass an array of structures —
// same sequence enables transitions
viewer.load({
  structures: [
    {
      label:     'MFE fold',
      sequence:  SEQ,
      structure: STR_1,
    },
    {
      label:     'Alternative A',
      sequence:  SEQ,
      structure: STR_2,
    },
    // ...
  ],
});

// Switch programmatically
viewer.switchToStructure(1);

// Or by label string
viewer.switchToStructure(
  'Alternative B'
);

// Read current index
viewer._currentStructIdx; // → 0
